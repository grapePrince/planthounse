
다음은 3차 발표를 진행하겠습니다.

3차 프로젝트의 경우 각종 식물을 판매하는 가상의 쇼핑몰을 가정하고 사이트를 제작하였습니다. 

쇼핑몰 사이트이기 때문에 
오늘의 집, 화해, 더가든 등의 쇼핑몰 사이트들을 위주로 분석을 진행하였고 
분석 내용을 토대로 모든 측면에서 우수한 사이트를 만들 수 있도록 목표를 세웠습니다. 

오늘의집 사이트의 구현 목표로는 아름다운 디자인과 사용자 경험을 높이는 것을 목표로 잡았습니다. 

오늘의집 사이트 역시 스토리 보드를 작성하여 기획을 진행하였고
스타일 가이드를 작성하여 디자인을 하였습니다. 
자세한 내용은 구현에서 말씀드리도록 하겠습니다. 

개발적인 측면에서 
웹표준을 준수하고자 노력하였고 
SCSS를 활용하여 코드를 깔끔하게 정리해주었습니다. 
작은 아이콘 및 로고 등은 스프라이트 처리를 해주었고요. 
OG 태그를 넣어주었습니다. 

그럼 바로 구현 내용을 발표하도록 하겠습니다. 


먼저 사이트는 모바일 전용으로 구현하였기 때문에 PC에서 들어갔을 때 이런 식으로 틀 안에 가둬주었습니다.
그래서 갤럭시 버전으로 들어가게 되면 틀이 사라지고 바로 페이지로 들어가게 됩니다. 

사이트는 전체적으로 나눔스퀘어, 나눔바른고딕 폰트를 사용하였고 
식물을 연상시키는 초록색을 포인트 컬러로 잡았습니다. 

헤더쪽은 좌측 메뉴를 눌렀을 때 스르륵 열리게 되고요. 
검색 메뉴는 검색어를 입력할 수 있고 reset 버튼이 있어서 바로 입력을 지울 수 있습니다. 
검색버튼을 누르거나 지난 검색어를 클릭하면 검색 목록이 뜨게 됩니다. 

위쪽의 헤더 부분은 스와이프 동작을 통해 옆으로 넘겨 볼 수가 있고요. 
클릭하면 해당 페이지로 이동합니다. 

아래는 배너가 있어서 슬라이드 방식으로 구현해주었고요. 

카테고리는 누르면 해당 카테고리의 페이지로 이동합니다. 
카테고리 목록의 경우 검색 목록과 비교했을 때 브레드스크럼이 있다는 점이 다른 점이고 
나머지는 동일한 UI 로 구현해 주었습니다. 

목록은 리스트 보기 방식을 1컬럼, 2컬럼으로 바꾸어 볼 수가 있는데요. 
CSS 만으로 구현한 것이라 페이지 로딩 없이 리스트 보기 방식을 바꿀 수 있습니다. 
옆의 셀렉트 박스는 사이트 디자인에 맞춰서 커스텀을 해주었습니다. 

메인 화면으로 돌아와서 아래쪽은 세일 섹션이고요. 
세일이 주다 보니 몇퍼센트나 할인을 하였는지 강조해서 리스트를 만들어 주었습니다. 
리스트 마다 들어가있는 찜하기 버튼의 경우 누르면 바로 찜하기가 가능하도록 구현하였고요. 

아래 신상품의 경우 크게 보여주고 싶다고 생각해서 이미지가 조금 크게 보여지도록 하였고. 
목록은 스와이프로 오른쪽으로 넘겨볼 수 있는데 넘길 수 있다는 것을 알려주기 위해서
조금 가려주었습니다. 
이런 식으로 찜하기 버튼이 사진 위쪽으로 올라가는 경우가 있는데 
이경우에는 버튼 주변으로 하얗게 바탕을 깔아주어 버튼을 보기 쉽게 해주었습니다. 

다음은 인기상품 목록인데요. 
품절/신규/인기 등의 태그가 붙을 수도 있고 없을 수도 있고 
할인 정보가 있을수도 없을 수도 있어서 각 경우의 수에 맞춰서 전부 대응할 수 있도록 
유연하게 CSS 를 잡아주었습니다. 

사이트 전체적으로 보시면 목록이 여기 저기에 중복되고 있는데요. 
각 목록마다 생긴 모습을 조금씩 다르게 해주어서 심심하지 않게 해주되 
목록 구현은 공통 부분을 동일한 CSS 를 사용할 수 있도록 모듈화에 신경을 써주었습니다. 

다음은 제품 상세 페이지입니다. 
제품 상세페이지 역시 브레드스크럼을 넣어주어서 제품이 속한 카테고리를 알 수 있도록 해주었고요. 
제품 이미지는 슬라이드를 이용해 넘겨볼 수 있도록 하였습니다. 

상세 쪽에 할인 정보가 있는 경우가 있는데요. 
이경우 현재 남은 할인 시간을 카운트다운 해주어서, 구매를 유도할 수 있게 해주었습니다. 
아래는 기타 상품에 관한 요약 정보이고요. 

상품 상세 페이지에는 위쪽에 헤더가 있어서 
상품 상세정보나, 리뷰 정보, Q&A 정보를 페이지 로딩 없이  눌러서 볼 수 있도록 하였습니다. 
상품 상세 정보는 구매에 연관된 주요한 화면이라 길게 작성을 해두었고요. 

각 탭마다 아래쪽에는 주문하기, 담기 버튼과 관련 상품 목록이 있어서 
구매에 활용할 수 있습니다. 

주문하기 레이어의 경우 옵션을 선택할 수가 있고요.
플러스, 마이너스 버튼을 누르면 개수가 올라가게 됩니다. 

다음 마이페이지는 각종 정보들을 볼 수 있는 페이지이고 메뉴로도 이동할 수 있습니다. 

장바구니 화면의 경우 각 상품들을 선택해서 주문을 진행할 수 있는데요. 
이런 식으로 체크박스를 전부 체크하게 되면 전체 선택 체크박스가 선택되게 되고 
하나라도 풀면 전체 선택 체크박스가 자동으로 풀리게 되어있습니다. 
전체선택 체크박스를 누르면 다른 체크박스들을 전부 선택/해제할 수 있습니다. 

장바구니 쪽에서도 플러스, 마이너스 버튼을 각각 이용할 수 있고요. 

찜한 목록의 경우 장바구니와 UI 가 유사한데요. 
장바구니 담기, 삭제하기, 주문하기 등을 아이콘과 버튼으로 동작을 바로 할 수 있습니다. 
주문하기 버튼의 경우 중요한 버튼이라서 조금 크게 넣어주었습니다. 

로그인 화면의 경우 심플하게 구현하였고 
아무것도 입력하지 않았을 땐 유효성 체크를 하게 됩니다 .

회원가입 화면 역시 유효성 체크를 해주며
기본정보 입력을 끝내면 아래 버튼이 활성화가 되게 됩니다. 

이렇게 가입버튼을 누르면 메인으로 돌아옵니다. 


그럼 이것으로 최종 발표를 마치겠습니다. 

감사합니다.
